FAQ

Quick answers before installing.

Does the download include the client?

Yes. The Flow launcher package includes the Flow Client mod jar. Minecraft libraries and assets are still downloaded from official Mojang and Forge sources at setup time.

Is OptiFine included?

No. OptiFine is paused for public stability and is not redistributed. The current public path focuses on a stable vanilla Forge runtime first.

What Minecraft version is supported?

Flow is focused on Minecraft 1.8.9 right now. Future versions are roadmap items, not current promises.

Is Flow affiliated with Mojang, Lunar, or Badlion?

No. Flow is an independent client project with original branding inspired by modern client UI patterns.
